# Promptify

Promptify is a NextJS application where the user can create, view, update and delete prompts. The user can also view other people's profiles where their prompts will be displayed. The application also provides search functionality allowing users to search by prompts, usernames and tags.


# Tech Stack
- NextJS
- TailwindCSS
- MongoDB

# Installation

1. Clone the repo
> git clone 
2. Go to the folder promptify
> cd promptify
3. Install dependencies
> npm install
4. In .env file, place your mongodb url
5. Go to console.google.com, create your OAuth client credentials and add them to .env file
6. Also add nextauth url, nextauth auth internal url and nextauth secret.
7. Run the application locally.
> npm run dev
